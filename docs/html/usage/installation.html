Installation¶

You can download FlaskS3Viewer executable and binary distributions from PyPI

Support versions¶

Type Name Version
Runtime Python >=3.7
Library boto3 >=1.12.2
Library click >=7.1.1
Library Flask >=1.1.1

Using pip¶

pip install flask_s3_viewer
